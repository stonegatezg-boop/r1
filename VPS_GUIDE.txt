============================================================

PREDUSLOVI:
-----------
[x] Python 3.8+ instaliran
[x] MT5 instaliran i ulogiran na broker racun
[x] Anthropic API kljuc (sk-ant-...)


BRZA INSTALACIJA (3 koraka):
============================

1. Kopiraj cijeli Clawder folder na VPS
   Primjer: C:\Clawder\

2. Pokreni kao Administrator:
   - Desni klik na VPS_INSTALL.bat
   - "Run as administrator"
   - Odaberi opciju 1 (Kompletna instalacija)
   - Unesi API kljuc kad te pita
   - Odaberi "1" za Windows Service (preporuceno)

3. U MT5:
   - F4 -> otvori MetaEditor
   - Otvori Clawder_Bridge.mq5
   - F7 -> Compile
   - Drag & drop Clawder_Bridge na XAUUSD M5 chart
   - Ukljuci AutoTrading (zeleni gumb gore)


DETALJNA OBJASNJENJA:
=====================

OPCIJA 1: Windows Service (PREPORUCENO za VPS)
----------------------------------------------
Prednosti:
  - Radi u pozadini bez CMD prozora
  - Automatski se pokrece pri restartu VPS-a
  - Automatski restart ako se srusi
  - Profesionalno rjesenje

Kontrola:
  nssm start ClawderBrain      # Pokreni
  nssm stop ClawderBrain       # Zaustavi
  nssm status ClawderBrain     # Status
  nssm remove ClawderBrain confirm  # Ukloni


OPCIJA 2: Task Scheduler
------------------------
Prednosti:
  - Pokrece se automatski pri loginu
  - Ne treba admin prava nakon instalacije
  - Jednostavnije za debugging

Kontrola:
  schtasks /run /tn ClawderBrainStartup    # Pokreni
  taskkill /im python.exe /f               # Zaustavi
  schtasks /delete /tn ClawderBrainStartup /f  # Ukloni


OPCIJA 3: Batch File (rucno)
----------------------------
  - Dvostruki klik na run_clawder.bat
  - CMD prozor mora ostati otvoren
  - Automatski restart svakih 10 sekundi ako padne

TROUBLESHOOTING:
================

1. Service se ne pokrece?
   - Provjeri logs/service_stderr.log
   - Provjeri je li API kljuc ispravan u .env

2. MT5 ne izvrsava odluke?
   - Provjeri je li AutoTrading ukljucen
   - Provjeri logs/clawder_*.log
   - Provjeri MinConfidence (default 0.7)

3. Nema CSV fajlova?
   - Provjeri je li EA ispravno kompajliran
   - Provjeri MT5 Experts tab za greske
   - Restartaj MT5

4. API greske?
   - Provjeri je li kljuc validan na console.anthropic.com
   - Provjeri internet konekciju
   - Clawder ce HOLD ako API ne radi (sigurno)


SIGURNOSNE PROVJERE (KILL SWITCH):
==================================

Clawder automatski PAUZIRA trading ako:
  - Dnevni gubitak > 3%
  - Spread > 30% ATR-a
  - Vise od 5 trade-ova dnevno
  - API timeout (default: HOLD)
